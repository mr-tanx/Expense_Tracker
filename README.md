💸 ExpenseTracker (React Native + Expo)

A simple, reliable personal expense tracking mobile application built using React Native (Expo).
Designed for daily real-world usage, with accurate balances and persistent local storage.

🚀 Features
💰 Balance Management

Set opening balance (Cash & Online separately)

Real-time balance updates after every transaction

Shows:

Cash balance

Online balance

Total available balance

🧾 Transactions

Add expenses (debit) or income (credit)

Choose Cash or Online mode

Each transaction shows:

Title

Date & time

Mode (Cash / Online)

Amount

Running balance after that transaction

Recent transactions appear on top

📄 Export & Reset

Export all transactions as PDF

Reset app with confirmation

Option to export before reset

💾 Persistence

Uses AsyncStorage

Data remains intact after:

App restart

Phone restart

App update (same package)

🛠️ Tech Stack

React Native

Expo

AsyncStorage

JavaScript (ES6+)
